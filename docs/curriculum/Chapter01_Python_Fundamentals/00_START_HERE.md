# Phase 1 — Python Fundamentals

**Goal:** Be able to write small Python scripts that read input, process data, and produce output.

## 📚 Lessons (do them in order)

| # | Lesson | File |
|---|--------|------|
| 1 | Hello World & how Python runs | `01_hello_world.py` |
| 2 | Variables & data types | `02_variables_and_types.py` |
| 3 | Strings & f-strings | `03_strings.py` |
| 4 | Numbers & math | `04_numbers_and_math.py` |
| 5 | Lists | `05_lists.py` |
| 6 | Dictionaries | `06_dictionaries.py` |
| 7 | Conditionals (if/elif/else) | `07_conditionals.py` |
| 8 | Loops (for / while) | `08_loops.py` |
| 9 | Functions | `09_functions.py` |
| 10 | Reading & writing files | `10_files.py` |

## 🏆 Mini-project at end of phase
**`mini_project_log_analyzer.py`** — Parse a log file, count error types, write a summary report.
(This mirrors what IT teams actually do daily.)

## ▶️ How to run a lesson
Open a terminal in this folder and run:
```powershell
python 01_hello_world.py
```
